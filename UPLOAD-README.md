# M1.5.2 首屏与响应式增量上传说明

本目录以 **M1.5.1 已上传完成** 为前提，只覆盖本轮产品体验相关文件。

## 上传方式

把上传包中的文件按原有相对路径覆盖到仓库根目录：

- `src/...` → 仓库 `src/...`
- `tests/...` → 仓库 `tests/...`
- `docs/...` → 仓库 `docs/...`
- `README.md`、`UPLOAD-README.md` → 仓库根目录

不要删除仓库中的其他文件。本包不包含 `node_modules`、`dist` 或测试截图，线上构建应由仓库源码重新执行。

## 本批内容

1. 首次打开不再自动注入示例对象，核心工作流从“生成 / 导入”开始。
2. 空工作台提供“AI 生成模型 / 导入模型 / 打开示例”三个入口。
3. 示例场景改为用户显式打开，继续复用原有 5 个验收对象。
4. ≤1366px 首次进入默认收起检查器与 AI/历史区。
5. 窄屏检查器和 AI/历史改为覆盖抽屉，不再继续压缩中央视口。
6. AI 输入框和服务状态文案产品化，视口工具与快捷提示在窄屏渐进收纳。
7. 增加 M1.5.2 永久版本记录与响应式默认布局测试。
