## 产品目标

补上“拓扑水密不等于几何健康”的证据缺口，并让用户在确认修复前看清具体新增和删除了什么。本版不扩大自动修复权限，不允许 Agent 或检查器直接改写复杂拓扑。

## 新增体验

1. 打印检查新增自交、内部壳和孤立碎片只读诊断。
2. 自交命中为错误；内部壳和孤立碎片为警告，并明确说明也可能是设计意图。
3. 大网格的自交扫描受三角形数和候选对预算保护；未完整覆盖时显示“部分检测”，不把零命中伪装为通过。
4. 信息条目增加连通壳数量，帮助区分单体模型、多零件集合和疑似碎片。
5. 复杂拓扑条目显示“只读诊断”标识，不出现自动修复按钮。
6. 修复预览增加“修复后叠加 / 仅看变化”切换。
7. 新增面使用绿色、删除面使用红色；原模型仍保持不变，确认后才生成派生副本。
8. 新诊断证据进入 AI 拆件输入协议，但 Agent 权限仍保持只读。

## 检测口径

- 连通壳：有效三角形通过共享边形成的面连通分量。
- 内部壳：一个闭合连通壳的包围盒严格位于另一闭合壳内部，且射线奇偶测试确认其采样点位于外壳内；结果标注为“疑似”，要求用户确认设计意图。
- 孤立碎片：排除内部壳后，相对主体面数或空间尺度显著较小的断开分量。
- 自交：不共享焊接顶点的三角形经过包围盒粗筛和三角形相交测试后产生的确定命中。
- 大模型保护：连通壳完整分析上限为 250,000 面；自交最多抽取 60,000 个三角形、测试 500,000 个候选对并记录 200 个命中；触发任一上限即标记为部分检测。

## 明确边界

- 部分检测未命中不表示模型无自交。
- 内部壳和孤立碎片可能是有意的多零件设计，本版只提供证据，不自动删除或合并。
- 不修复任意自交、内部壳、非流形结构或复杂布尔残片。
- 不替代切片器的切片路径、支撑和实体布尔验证。
- Agent 只能读取新证据并给出建议，不能调用写模型工具。

## 自动化验证

- 单一封闭立方体：1 个闭合壳，零自交、内部壳和孤立碎片。
- 嵌套封闭立方体：2 个闭合壳，识别 1 个疑似内部壳，零表面相交。
- 主体外单独三角片：识别 1 个孤立碎片和对应面数。
- 两片不相邻三角形穿过：识别 1 组自交。
- 超过扫描预算：明确返回部分检测。
- 修复删除退化面：返回红色删除面几何；拒绝路径不返回差异几何。

## 手动验收清单

1. 运行打印检查，确认空态文案包含自交、内部壳和孤立碎片。
2. 对复杂问题条目确认出现“只读诊断”，且没有“修复预览”按钮。
3. 对开口盒进入修复预览，确认出现“修复后叠加 / 仅看变化”切换。
4. 确认绿色表示新增面、红色表示删除面；切换显示方式不写历史、不改变资产。
5. 点击“生成修复副本”后仍沿用 M1.7 的派生资产、自动复检和撤销闭环。
6. 对超大模型确认出现部分检测提示，文案包含已扫描面数和总面数。
