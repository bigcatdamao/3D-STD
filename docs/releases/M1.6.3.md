## 产品体验变化

1. Agent 返回的下一步保留为结构化步骤，不再退化成一串无法操作的文字。
2. 新增“立即可做”区域：定位首个打印问题、查看打印检查、重新检查证据。
3. 定位只改变选择、高亮和相机，不写历史；重新检查只刷新确定性诊断。
4. 切割预览和朝向分析继续显示“阶段二”禁用状态，不制造尚未实现的能力错觉。
5. 场景或打印检查证据变化都会使旧建议过期，避免用户拿旧结论继续操作。
6. 真实结果显示响应耗时和总 token，便于用户感知等待，也为评测和成本校准提供数据。
7. 线上首轮四视角验收暴露 45 秒超时；本版将 Worker 等待窗口调整为 75 秒、浏览器调整为 85 秒。延长窗口只用于避免误降级，p95 ≤ 45 秒仍是后续优化门槛。

## Gold Set 底座

- 新增 `split-analysis-gold.v1` JSON 契约和 24 样本目标。
- 首批建立 5 个种子定义：整体可打印、尺寸超限、网格阻断、支撑敏感、外观面敏感。
- 自动评分覆盖结论、阻断原因、证据引用、建议件数、能力限制和方案结构。
- 人工评分覆盖方案可用性、行动性、表面风险诚实度、装配清晰度和幻觉。
- 关闭门槛：结论准确率 ≥ 85%、证据完整率 ≥ 95%、幻觉率 ≤ 5%、p95 ≤ 45 秒。

## 权限边界

- 第一阶段 Agent 仍然没有工具调用与模型写权限。
- “立即可做”只复用已验证的本地选择、相机与打印检查能力。
- 不提供应用切割、自动沉底或自动修复按钮；阶段二预览和阶段三写工具保持关闭。
- API Key 不进入遥测、Gold Set、前端或 Git。
